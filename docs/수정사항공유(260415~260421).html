✨ 새 기능

대회 기록실

기록 탭에 🏅 기록실이 추가됐습니다. 참가한 마라톤·달리기 대회 기록을 직접 저장하고 모아볼 수 있습니다.

  • 종목 선택: 5K / 10K / 하프 / 풀 / 기타
  • 대회명, 날짜, 완주 기록 입력 (숫자만 쳐도 자동 포맷: 34215 → 3:42:15)
  • 인증 사진, 공식 기록 링크, 메모 첨부 가능
  • 종목별 개인 최고 기록(PB)을 자동으로 감지해 🏆 배지 표시

클럽 마일리지 탭 — 합산 표현 게이지

클럽 마일리지 탭에서 이번 달 클럽 합산 거리를 게이지로 시각화했습니다.

서울-부산 왕복, 백두대간 종주 등 실제 거리에 빗댄 달성 배지(×N)와 다음 마일스톤까지의 진행률을 한눈에 확인할 수 있습니다.

🔧 기능 개선

카카오톡 공유 "나도 기록하기" — 클럽 초대 연계

클럽 피드의 운동 기록을 카카오톡으로 공유했을 때, 링크를 받은 사람이 나도 기록하기 버튼을 누르면 해당 운동 상세 페이지로 바로 이동합니다.

클럽에 아직 가입하지 않은 경우 클럽 가입 페이지로 자동 연계되며, 가입 완료 후 원래 운동 기록 페이지로 복귀합니다.

누구의 기록인지 확인할 수 있도록 상단에 OOO님의 기록 배너도 표시됩니다.

기록 탭 통계 개선

기록 탭 기본 화면이 통계 탭으로 변경됐습니다.

  • 최근 6개월 운동일수 또는 달린 거리 추이를 바 차트로 확인
  • 막대를 누르면 해당 월 상세 통계로 이동
  • 달린 거리 차트에서는 종목(달리기/트레드밀 등) 선택 가능
  • 전달 대비 +10% / +20% 목표선 표시
  • 종목별 이번 달 거리와 달성률(전달 대비 %) 게이지로 표시

피드백 접수창 모바일 UX 개선

더보기 → 수정 요청 / 버그 제보 화면을 좀더 직관적으로 바꿨습니다.

🐛 버그 수정

다른 사람의 기록 상세 화면에 수정/삭제 버튼이 간헐적으로 노출되던 문제

카카오 공유 링크로 진입한 타인의 운동 기록에서 수정·삭제 버튼이 간헐적으로 노출되던 오류를 수정했습니다.

본인 기록일 때만 수정/삭제 버튼이 나타나도록 처리했습니다.

피드백·버그 제보 전송이 안 되던 문제

더보기 → 수정 요청 / 버그 제보를 작성해도 "전송에 실패했습니다"가 뜨며 메일이 발송되지 않던 오류를 수정했습니다.

불편한 점이 있으시면 더보기 → 수정 요청 / 버그 제보로 알려주세요. 감사합니다!
