# 수정사항 (2026-04-08)

## 1. 리더보드 검색 기능 추가
- 마일리지 탭 상단에 돋보기(🔍) 버튼 추가
- 클릭 시 검색창 열림 → 닉네임으로 멤버 검색 가능
- 검색 결과: 해당 멤버를 중심으로 ±5명 표시, 핑크 하이라이트
- X 버튼으로 검색 초기화 → 원래 순위표로 복귀

## 2. 리더보드 전체 보기 기능 추가
- 기존: 상위 5명만 표시 후 생략
- 전체 보기 버튼 클릭 시 전체 순위 펼쳐짐
- 다시 접기 버튼으로 축소 가능

## 3. 멤버 운동 기록 상세 → 뒤로가기 탭 유지
- 리더보드에서 멤버 카드 클릭 → 운동 기록 상세 페이지 진입
- 뒤로가기(`<`) 클릭 시 마일리지(리더보드) 탭으로 복귀
- 기존: 피드 탭으로 초기화되던 문제 수정

## 4. 4월 리더보드 미표시 버그 수정
- 클럽 내 운동 기록이 많은 경우 리더보드가 아예 표시되지 않던 문제 수정

## 5. 계단 운동 단위 표시 버그 수정
- 운동 기록 저장 시 단위가 잘못 저장되던 문제 수정
- 증상: 계단 - 시간으로 30을 입력했는데 저장 후 "30층"으로 표시됨
- 원인: 저장 시 세부 항목(시간/층수) 단위가 아닌 종목 기본 단위(층)가 저장됨
- 수정: 항상 선택한 세부 항목의 단위를 저장하도록 변경

### 계단 마일리지 계산 기준
계단 운동은 **시간(분)** 또는 **층수** 두 가지 방식으로 기록 가능하며, 각각 별도 계수 적용.

| 입력 방식 | 예시 | 환산 | 마일리지 |
|---|---|---|---|
| 시간(분) | 30분 | — | 5점 |
| 층수 | 100층 | — | 5점 |

- 시간 기준: **6분 = 1마일리지**
- 층수 기준: **20층 = 1마일리지**
- 30분 운동 = 100층 오르기와 동일한 5마일리지로 인정
