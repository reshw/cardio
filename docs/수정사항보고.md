## 이번에 수정된 항목 (총 6건)

---

### 1. 기록 탭 통계 — 월 선택 기능 추가

이번 달만 보이던 통계에 월 이동 기능을 추가했습니다.
`‹ 2026년 4월 ›` 형태로 좌우 이동이 가능하고, 가운데를 누르면 연/월을 직접 선택할 수 있습니다.

---

### 2. 클럽 상세 통계 — 선택 월 연동

클럽 화면에서 3월로 이동 후 상세 통계를 열면 이번 달이 떠있던 문제를 수정했습니다.
이제 선택한 달 기준으로 상세 통계가 열립니다.

---

### 3. 클럽 상세 통계 — 운동일수 0 버그 수정

운동별 상세 통계에서 운동일수가 항상 0으로 나오던 버그입니다.

원인이 두 가지였습니다.
- 날짜 데이터 자체가 저장되지 않고 있었음
- 한국 시간 기준이 아닌 세계 표준시 기준으로 저장되어 자정 전후 운동의 날짜가 어긋남

두 가지 모두 수정했습니다. 신규 등록 운동부터 정상 집계됩니다.
기존 데이터는 클럽 메뉴의 마일리지 재계산을 실행하면 반영됩니다.

---

### 4. 클럽 피드 — "n번째 운동" 기준 변경

카톡 공유 시 붙는 "오늘 클럽 n번째" 문구 기준을 변경했습니다.

- 기존: 운동한 시각 순서 → 뒤늦게 올린 운동이 중간에 끼어들면 기존 번호가 밀림
- 변경: 앱에 등록한 순서 → 한번 부여된 번호는 이후 변경되지 않음

---

### 5. 클럽 랭킹 — 일반 회원 탭 구분 표시

전체 탭과 일반 회원 탭의 스크린샷이 구분이 안 된다는 점을 감안해,
일반 회원 탭에서 마일리지 아래 **(일반)** 텍스트를 작게 표시하도록 했습니다.

---

### 6. 클럽 랭킹 — 표시 범위 조정

- 내 순위 20위 이내: 전체 표시
- 내 순위 20위 밖: 상위 5위 + 내 주변 ±3명 표시, 나머지 중략 처리

기존 Top 10 고정 방식에서 변경된 내용입니다.
하위권 유저가 내 순위를 확인하려면 스크롤을 많이 내려야 했던 구조적 불편을 개선했습니다.
