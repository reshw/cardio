# (검토) 함께 운동한 사람 등록 기능

## 개요
클럽 피드에서 내 운동 기록에 함께 운동한 사람을 태그하는 기능.
상대방 기록에는 영향 없음 - 내 피드/클럽 표시용.

## 채택 방향
- 운동 기록 추가 시점이 아닌, **클럽 피드 카드에서 사후 추가** 방식
- 클럽 맥락이 확정된 상태에서 해당 클럽 멤버만 선택 → 닉네임 혼동 없음

## 흐름
1. 클럽 피드 내 내 카드에만 "함께한 사람 +" 버튼 노출
2. 해당 클럽 멤버 목록(클럽 닉네임) 팝업에서 선택
3. 카드에 `함께: 닉네임, 닉네임` 표시

## DB
```sql
ALTER TABLE workouts ADD COLUMN with_user_ids uuid[] DEFAULT NULL;
```

## 변경 필요 파일
- `workoutService.ts` - Workout 인터페이스 + updateWithUsers() 메서드
- `feedService.ts` - WorkoutFeedItem에 with_users 필드
- `clubService.ts` - getClubWorkoutFeed()에서 with_user_ids → 클럽 닉네임 resolve
- `WorkoutFeedCard.tsx` - 버튼 + 멤버 선택 UI + "함께:" 표시
- `WorkoutDetailModal.tsx` - 상세에서도 표시
