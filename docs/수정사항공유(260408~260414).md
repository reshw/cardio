## 🐛 버그 수정

**운동 기록 저장 시 단위가 잘못 저장되던 문제**
- 계단 운동에서 "시간(분)"으로 입력했는데 "층"으로 저장되는 오류를 수정했습니다.

**운동 기록 수정 시 "실패" 알림이 잘못 뜨던 문제**
- 실제로는 수정이 정상적으로 완료됐는데 "수정에 실패했습니다" 알림이 뜨던 오류를 수정했습니다.

**운동 기록 수정 시 날짜/시간이 9시간 어긋나던 문제**
- 수정 모드에서 시간이 실제보다 9시간 빠르게 표시되던 문제를 수정했습니다.

**4월 리더보드가 표시되지 않던 문제**
- 클럽 내 운동 기록이 많을 경우 리더보드가 아예 나오지 않던 문제를 수정했습니다.

## ✨ 새 기능

**멤버 운동 기록에서 좋아요 · 댓글 가능**
- 클럽 마일리지 탭에서 멤버 이름을 클릭하면 나오는 운동 기록 화면에서, 각 운동 카드에 좋아요와 댓글을 바로 남길 수 있습니다.
- 댓글 버튼을 누르면 해당 카드 아래에 댓글창이 펼쳐지며, 답글도 가능합니다.

**피드백 · 버그 제보 기능**
- 더보기 메뉴에 **수정 요청 / 버그 제보** 항목이 추가됐습니다.
- 제목과 내용을 작성 후 전송하면 운영자에게 바로 전달됩니다.

## 🔧 기능 개선

**오늘의 운동 날짜 이동이 더 편해졌어요**
- 날짜를 누르면 달력이 열려 원하는 날짜로 바로 이동할 수 있습니다.
- 오늘이 아닌 날짜를 보고 있을 때 **오늘** 버튼이 나타나 한 번에 돌아올 수 있습니다.

**리더보드 멤버 검색 개선**
- 검색창에서 멤버를 검색하면 해당 멤버만 결과로 표시됩니다.
- 선택 후 리더보드에서 해당 멤버 앞뒤 순위가 함께 보여 내 위치를 파악하기 쉬워졌습니다.
- 모바일에서 검색 시 키보드가 올라와도 검색창이 가려지지 않도록 개선됐습니다.

**멤버 운동 기록 — 운동일수 표시**
- 기존 "총 운동 N회" 표시가 **운동일수 N일**로 변경됐습니다.
- 같은 날 여러 번 운동해도 1일로 집계되며, 클럽에서 인정하지 않는 종목은 제외됩니다.
